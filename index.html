Dublin TypeScript Meetup

TypeScript is a typed superset of JavaScript that compiles to plain JavaScript. In this Meetup, we teach and learn effective ways to use Typescript, share cool projects we've built using the language, and do hands-on practice projects together.

All skill levels are welcome. Come hang out and share your enthusiasm for modern web technologies!

 

Join us!

Are you a web developer that is interested in learning more about Microsoft's open source TypeScript language? Are you already using TypeScript and want to meet other TypeScript developers? This is the group for you.

Please visit www.meetup.com to sign up for upcoming events.

Alternatively, you can follow us on twitter at @DubTypeScript to learn more about the latest TypeScript news and events.
